# 📝 To-Do List Web App

A simple and responsive To-Do List application built using **HTML, CSS, and JavaScript**. This app helps users manage daily tasks efficiently with features like task filtering, editing, dark mode, and persistent storage.

---

## 🚀 Features

- Add new tasks
- Edit existing tasks
- Delete tasks with confirmation
- Mark tasks as completed/incomplete
- Filter tasks:
  - All
  - Completed
  - Pending
- Dark Mode toggle (saved in localStorage)
- Data persistence using localStorage
- Press "Enter" to quickly add tasks
- Responsive design for mobile devices
- Custom popup notifications (no default alerts)

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, Media Queries)
- JavaScript (DOM Manipulation, Local Storage)
- Boxicons (for icons)
